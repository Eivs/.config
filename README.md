### 基础工具
- vim/nvim - 文本编辑器
- curl/wget - 网络下载工具
- git - 版本控制系统
- htop/top - 系统监控工具
- tmux/screen - 终端复用器
- zellij - 终端复用工具
- zip/unzip - 压缩/解压缩工具
- net-tools - 网络工具集合

### 网络工具
- dnsutils - DNS查询工具
- iputils-ping - ping工具
- traceroute - 路由跟踪工具
- nmap - 网络扫描工具
- tcpdump - 网络数据包分析工具
- netcat - 网络工具
- telnet - 远程登录工具
- mtr - 网络诊断工具

### 系统工具
- lsof - 列出打开文件
- rsync - 文件同步工具
- tree - 目录树显示工具
- jq - JSON处理工具
- sysstat - 系统性能监控工具
- iotop - I/O监控工具
- iftop - 网络带宽监控
- nload - 网络流量监控
- atop/glances - 系统监控工具
- ncdu - 磁盘使用分析工具
- btm 系统监控

### 文件搜索工具
- fzf - 模糊查找工具
- bat - 增强版cat
- fd-find - 替代find命令
- ripgrep - 替代grep命令

### 安全工具
- fail2ban - 防暴力破解工具
- ufw/firewalld - 防火墙

### k8s 工具
- k9s k8s 集群管理工具
- lazydocker 容器管理工具

```bash
apt update
apt install zsh vim curl wget ripgrep fd-find screen bat git tmux zip net-tools dnsutils iputils-ping traceroute tcpdump telnet mtr lsof rsync jq tree sysstat iotop iftop btm htop
```
